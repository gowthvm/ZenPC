### üî¥ Hard Compatibility (10 checks - ERRORS ONLY)
Build-breaking incompatibilities that prevent assembly or operation:

**Physical Constraints**
- GPU ‚Üî Case clearance (length, height)
- Cooler ‚Üî Case height clearance
- Motherboard ‚Üî Case form factor
- PSU ‚Üî Case form factor

**Electrical Constraints**
- CPU ‚Üî Motherboard socket
- RAM ‚Üî Motherboard memory type
- Cooler ‚Üî CPU socket compatibility

**Power Constraints**
- GPU power connectors (8-pin, 6-pin, 12VHPWR)
- Motherboard power (24-pin)
- CPU power (4-pin, 8-pin)

### ‚ö†Ô∏è Performance Warnings (5 checks - ADVISORY)
Suboptimal configurations with clear trade-offs:

- RAM speed downclocking (~5% performance loss)
- PSU low headroom (<30% efficiency risk)
- CPU/GPU bottleneck (15-30% gaming FPS loss)
- Cooler underpowered (thermal throttling risk)
- PCIe generation mismatch (~5-20% bandwidth loss)

### ‚ÑπÔ∏è Informational (5 checks - EDUCATIONAL)
Best practices and future planning:

- PCIe backward compatibility explained
- ECC RAM consumer board behavior
- NVMe speed limitations
- PSU modularity cable management
- Socket upgrade path planning
