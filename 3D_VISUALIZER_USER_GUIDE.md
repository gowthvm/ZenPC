# 3D PC Visualizer - Quick User Guide

## What is the 3D PC Visualizer?

The 3D PC Visualizer is an interactive tool that lets you see your PC build in stunning 3D detail. You can rotate it, customize component colors, and explore how everything fits together.

## Accessing the Visualizer

### From Landing Page:
1. Scroll to the "See Your PC in 3D" section
2. Click "Try 3D Preview Now" button
3. Or scroll to "Sample Builds in 3D" to see example previews

### Direct Access:
Visit `/visualizer3d` in your browser

## Using the 3D Visualizer

### Basic Controls

**On the Left Side (3D View):**
- **Auto-Rotate**: Watch your PC spin automatically (default ON)
- **Zoom In**: Get a closer look at components
- **Zoom Out**: See the full picture
- **Reset**: Return to default view

### Customizing Colors

**On the Right Control Panel:**

1. **Component Colors**
   - **GPU** (Graphics Card): Click the color picker or enter hex code
   - **CPU** (Processor): Click the color picker or enter hex code
   - **RAM** (Memory): Click the color picker or enter hex code
   - **Storage** (SSD): Click the color picker or enter hex code

2. **Quick Presets**: One-click color schemes
   - Gaming Powerhouse (Red/Blue)
   - Professional Workstation (Orange/Amber)
   - Streaming Setup (Cyan/Purple)
   - Budget Builder (Grayscale)

3. **Reset Colors**: Return to default colors

### Settings

- **Auto Rotate Toggle**: Turn auto-rotation on/off for manual inspection

### Export Options

- **Download Image**: Save your 3D build as an image
- **Share**: Share your custom build with others
- **Save Build**: Save your configuration for later

### Component Colors Legend:
- **Orange Box** = PC Case (chassis)
- **Red/Custom** = GPU (Graphics Card)
- **Blue/Custom** = CPU (Processor)
- **Yellow/Custom** = RAM (Memory)
- **Green/Custom** = Storage (SSD)
- **Dark Gray** = Power Supply
- **Cables** = Orange routing

## Color Picker Guide

### How to Use the Color Picker:

1. Click the colored square next to any component
2. A color picker will appear
3. Click to select a color from the palette
4. Or enter a hex code directly (e.g., #FF0000 for red)

### Popular Hex Colors:

Gaming:
- Red: #FF0000
- Blue: #0000FF
- Green: #00FF00
- Purple: #800080

Professional:
- Silver: #C0C0C0
- Navy: #000080
- Gold: #FFD700
- White: #FFFFFF

RGB:
- Red: #FF0000
- Green: #00FF00
- Blue: #0000FF

## Frequently Asked Questions

**Q: Can I move individual components?**
A: Currently, this is a visualization tool. Component placement is fixed to show a realistic PC case layout.

**Q: Can I use this with my custom build?**
A: Yes! Use the color picker to match your PC's RGB lighting scheme.

**Q: How accurate is the model?**
A: The model shows realistic proportions and typical component sizes. Exact measurements may vary by model.

**Q: Can I export this to show others?**
A: Yes! Use the Share button to generate a link, or Download to save as an image.

**Q: Does this show thermal info?**
A: The current version shows component layout. Check Performance Estimates for thermal data.

**Q: What browsers support this?**
A: All modern browsers (Chrome, Firefox, Safari, Edge) with WebGL enabled.

## Troubleshooting

**Problem: 3D model not showing**
- Make sure you're using a modern browser
- Try refreshing the page
- Disable browser extensions temporarily

**Problem: Colors not changing**
- Make sure you're entering valid hex codes
- Try using the color picker instead of typing

**Problem: Model is laggy**
- Try zooming out to reduce detail
- Turn off auto-rotation
- Close other browser tabs

**Problem: Controls not working**
- Make sure "Show Controls" is enabled
- Try using a different input method (picker vs. text)
